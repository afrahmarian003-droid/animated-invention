    • Reminder on adding a new entry:

    1. Open posts.js
    2. Scroll to the bottom of the posts array
    3. use script below
    4. Fill it in
    5. Commit the changes

    Entry script

    {
  id: "unique-id-here",
  title: "Title of the Entry and Author",
  written: "YYYY-MM-DD",
  tags: ["optional", "tags"],
  excerpt: "A short one-line summary.",
  body: `
Write your entry here.

ALWAYS leave a blank line between paragraphs.
`
},

    Important Notes

    • Written date controls sorting order
    • Posts appear by when they were written, not uploaded
    • Each id must be unique
    • Do not remove commas between entries

    If something breaks or doesnt load, check for:

    • missing commas inside script
    • duplicate IDs
    • missing backticks (`)
