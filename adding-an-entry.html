Adding an Entry

All writing on this site lives in a single file: posts.js.

To add a new entry:

  1. Open posts.js
  2. Scroll to the bottom of the posts array
  3. Paste the template below
  4. Fill it in
  5. Commit the changes

Entry Template

{
  id: "unique-id-here",
  title: "Title of the piece",
  written: "YYYY-MM-DD",
  tags: ["optional", "tags"],
  excerpt: "A short one-line summary.",
  body: `
Write your entry here.

Leave a blank line between paragraphs.
`
},

Important Notes

  • Written date controls sorting order
  • Posts appear by when they were written, not uploaded
  • Each id must be unique
  • Do not remove commas between entries

If something breaks, check for:

  • Missing commas
  • Duplicate IDs
  • Missing backticks (`)
